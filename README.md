# pesquisa_agricola_municipal
Para reconstrução e limpeza das quatro variáveis de nível da PAM, todos os cultivos disponíveis, todos os anos disponíveis.
No mesmo repo, deve haver uma pasta data\raw com um arquivo .xlsx para cada ano, com todos os municípios do Brasil, de acordo com download do SIDRA (parámetros análogos a T/5457/N6/ALL/V/214,215,216,8331/P/LAST 1/C782/ALLXT/L/V,C782,T+P,MCTCTN). 
Esse arquivo deve apresentar uma planilha para cada variável: área plantada, área colhida, quantidade produzida, valor da produção. 
Preparamos um único arquivo limpo com tudo.
